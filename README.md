# laser-tag
small game with lasers, mirrors, and mazes
The game was developed with the objective of create an analytic game which could help people to test and improve
their attention and analysis skills.
The game was developed by Juan Fernando Martinez Dominguez, was programed with Java programming language 
under Object Oriented Paradigm and using recursive techniques and data structures like linked lists and Binary Search Tree
The game run under a Desktop environment and CLI
This game could be update in the future, so all suggestions and Ideas are good received
